# My nvim configs

**Required Requirements:**

_nvim >= 0.7_
_latest Packer_
_Lua_

**Install config:**

1. Install nvim
2. Install Packer
3. Copy files from nvim dir(lua direcotory + init.lua file) into `~/.config/nvim` (for Ubuntu <= 22.04)
4. Run nvim
5. Enter `Packer` command, and install all plugins
